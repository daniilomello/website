---
title: "Criando um novo projeto no Github via CLI"
publishedAt: "2024-06-18"
---

O Github tem uma [CLI](https://cli.github.com/) com comandos que facilitam a vida na hora de criar repositórios e iniciar novos projetos, pocs, etc. Você pode fazer a instalação [por aqui](https://cli.github.com/).

Com ele instalando, inicie o seu projeto novo com os seguintes comandos:

Inicie um projeto Git:

```bash
git init
```

Crie uma branch main para o repositório:

```bash
git checkout -b main
```

Crie um novo repositório:

```bash
gh repo create repo-name --public --source=. --remote=origin
```

- A flag `--public` cria o repo como público
- A flag `--source` define o caminho da pasta do repo
- A flag `--remote` define o nome de origin do repo


Adicione os arquivos ao staged:

```bash
git add .
```

Faça o commit dos arquivos:

```bash
git commit -m "feat: add initial react files"
```

Realize o push do projeto para a branch main:

```bash
git push -u origin main
```

Abra o repositório no navegador para conferir:

```bash
gh browse
```

Pronto, repositório criado com sucesso!
