The idea of dispose is to be used as a guide so you can quickly access the most importants commands on Git.

## Git Configuration

**Setting up an username and email:**
```bash
git config --global user.name "Danilo Mello"
```

```bash
git config --global user.email "danilo@email.com"
```

**Checking user configurations:**
```bash
git config user.name
```

```bash
git config user.email
```

```bash
git config --list
```

**Creating alias:**
```bash
git config --global alias.s status
```
