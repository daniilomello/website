## Introduction
Guide on how to update your project
    
## Prerequisites
Basic knowledge about npm

## Getting Started

01 - On the package.json hover over the lib you want to update to see the latest version
02 - Using vscode extension called versionlens to update
03 - npm outdated to check outdated versions
04 - Using the npx npm-check-updates package
